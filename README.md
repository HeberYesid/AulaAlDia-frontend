# AulaAlDia Frontend

Repositorio frontend (React + Vite) extraido desde el monorepo principal.

## Setup rapido

```powershell
pnpm install
copy .env.example .env
pnpm dev
```

Frontend local: http://localhost:5173

## Quality gate

```powershell
pnpm lint
pnpm build
pnpm test:integration
pnpm test:coverage
```
